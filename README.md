## xiaoshuo-native
react-native 做的小说阅读器

1. 首先得配置sdk,最好安装studio，具体配置可以看官网

2.构建Android测试
```react-native run android```

3.如果是只跑服务
```react-native start```

4.发布apk
> 首先生成key, 具体看官网
```cd android && ./gradlew assembleRelease```

*根目录下的app-release.apk 是可以直接安装使用的*
*注意：手机需要打开开发者模式*
